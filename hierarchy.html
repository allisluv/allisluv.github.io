Type Hierarchy - Spring 2016

This project intended to create an understanding of type hierarcy though layout, spacing, font size, font weight, font style, proximity of text.

Acheivements - I was able to come up with some intrigueing compositions. Dealing with the same content for each poster made it easier as I got more familiar with the way the content could be divided and highlighted.

Challenges - As I played wit layout and composition I lost sight of the main purpose of creating a type hierarchy for the content. I often started with a good composotion and had to work backward to emphasize the important parts of the content.
